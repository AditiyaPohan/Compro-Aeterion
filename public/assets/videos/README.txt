HERO BACKGROUND VIDEO
=====================

Letakkan video corporate building di folder ini dengan nama:

    corporate-building.mp4

Rekomendasi:
- Resolusi 1920x1080 (atau 1280x720), H.264/MP4, < 8 MB, ~15-25 detik, loop mulus.
- Tanpa audio (otomatis di-mute).
- Sumber bebas lisensi: Pexels Videos / Coverr / Mixkit (kata kunci: "corporate building", "modern office", "city skyscraper").

Sampai file video ditambahkan, Hero otomatis menampilkan poster
"hero-poster.webp" (foto gedung kaca korporat) + gradient brand sebagai
fallback, jadi tampilan tetap rapi dan profesional.

Tidak perlu ubah kode — komponen Hero sudah membaca path di atas
(components/sections/hero.tsx → <video src="/assets/videos/corporate-building.mp4">).
